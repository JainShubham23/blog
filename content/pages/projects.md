## Machine Learning & AI Projects

### Online News Popularity Predictor
**Technologies**: Python, Scikit-learn, Pandas, NumPy, Random Forest, Naive Bayes  
**Description**: ML models to predict the future popularity of online articles and optimize advertising strategies.  
**Key Achievements**:
- Implemented PCA for feature reduction and dimensionality optimization
- Developed and compared multiple models: Naive Bayes, Logistic Regression, KNN, Random Forest
- Added cost matrix analysis identifying $6.47M profitability with Random Forest model
- Focused on UX design for improved user interaction with prediction tool

[View on GitHub](https://github.com/JainShubham23/OnlineNewsPopularity)

### Graduate Admission Predictor
**Technologies**: Python, React, REST API, Machine Learning, GitHub Pages  
**Description**: Cross-platform application predicting university admission chances based on student credentials.  
**Key Features**:
- Tested and implemented 5 different ML models for optimal accuracy
- End-to-end REST API using Python with pickle files of trained models
- React frontend with responsive design hosted on GitHub Pages
- Comprehensive technical documentation and UX improvements

[View on GitHub](https://github.com/JainShubham23/college-predictor)

### Adversarial Attack Research
**Technologies**: Python, PyTorch, AlexNet, ResNet, CIFAR Dataset  
**Description**: Research project investigating clean label poisoning attacks on neural networks.  
**Research Focus**:
- Re-implemented "Poison Frogs! Targeted Clean-Label Poisoning Attacks on Neural Networks" paper
- Utilized AlexNet and ResNet architectures with CIFAR dataset
- Explored mitigation solutions for adversarial attacks
- Technical documentation and model performance testing

[View on GitHub](https://github.com/JainShubham23/AdverserialAttackNN)
